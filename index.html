IDs as bookmarks

An ID is placed in an element, then a link is made with an href attribute that calls the ID.
In the example below, the link is linked to the ID "#top-of-page", which was attributed to the first h1 element on this page.

Example; go to start
A bookbark is linked this way if it is on another page; go to start

Hashtag/pound sign # is use to signify an ID whereas fullstops/dots/points . signify a class

Classes

IDs are specific whereas classes signify groups that are to be styled.
A number sign # is used to recognise and ID while a dot . is used to recognise a class.
Multiple IDs and classes can be assigned to the same type of element specifically throughout a document.

Iframes

An iframe allows you to show another page within your page.
Iframes don't work for secured webpages.
By targetting an iframe in a link, the iframe source can be altered.
